Cohort-sized tutorial class for the module Hydrodynamic Stability in the MSc programmes.

Syllabus
======
* Stability Analysis
* Absolute and Convective instability
* Transient Growth
* Bifurcation analysis
* Linearisation and stability of Navier-Stokes equation

Teaching Format
======
Interactive, big-class tutorial session to go through problem sheets set by lecturer, with highlights on skills and techniques required to tackle problems in problem sheets and exams.
